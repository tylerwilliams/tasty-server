New tastes

Posting a new taste is easy; you can post a single taste or an array of tastes to the user specific endpoint and the server will respond with an array map of taste timstamp to boolean values. The value indicates whether or not the taste was successfully recorded on the server. If it was not, it's the client's responsibility to try again later.
